Declarative Deployments:
All cluster configurations are stored in Git. The cluster’s state should always match Git.

Continuous Reconciliation:
ArgoCD monitors the Git repository and automatically applies changes to the cluster (auto-sync) if it detects differences.

Drift Detection:
Any manual changes in the cluster (like scaling replicas or modifying a deployment) are detected as drift. With selfHeal enabled, ArgoCD would automatically revert these changes to match Git.
